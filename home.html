Lung Disease Prediction Web Application Using Image Texture Analysis in Orange

The Lung Disease Prediction Web Application utilizes image texture analysis to identify lung diseases from medical images. Built using Orange, a data visualization and analysis tool, the application preprocesses images to extract significant features, including texture and intensity.Users can upload images, which are analyzed in real-time, yielding predictions such as normal or lung opacity. The application aims to enhance diagnostic accuracy and provide timely health insights.
